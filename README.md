
# Memeverse

A nextjs app router project that handles meme generation, has a trending memes page and a page to explore different templates. User can create memes based on how many lines the meme templates handles.


## Tech Stack

NextJS, TailwindCSS, HeroUI, https://memegen.link


## Demo

https://memeverse-nine.vercel.app/


## Deployment

To deploy this project run

```bash
  npm i
  npm run deploy
```
